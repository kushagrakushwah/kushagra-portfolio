> *"Engineering Scalable, Robust & Secure Solutions."*

A futuristic, interactive personal portfolio website built with **React** and **TypeScript**. Designed with a **Cyberpunk/Terminal aesthetic**, featuring neon visuals, glitch effects, and a fully immersive user experience.

---

## 🚀 Features

### 🎨 Visuals & UI
* **Cyberpunk Aesthetic:** Dark mode interface with Neon Cyan (`#22d3ee`) and Purple (`#d946ef`) accents.
* **Holographic Profile:** Animated profile section with glowing borders and particle effects.
* **Custom Cursor:** A specialized neon-arrow SVG cursor for a high-tech feel.
* **Scroll Progress Bar:** Top-screen gradient indicator that tracks reading progress.
* **Custom Scrollbar:** Themed scrollbar to match the dark UI.

### ⚡ Interactivity
* **"Hacker" Text Decoding:** Section headers scramble and decrypt on hover (`HackerText` effect).
* **System Alerts:** Browser tab title changes when the user switches tabs ("⚠ System Alert: User Disconnected").
* **Typewriter Effects:** Dynamic text rendering for role descriptions.
* **Smooth Scrolling:** Custom navigation to handle smooth transitions between sections.

### 🛠 Tech Stack
* **Frontend:** React (Vite), TypeScript
* **Styling:** Tailwind CSS, Shadcn UI, Lucide Icons
* **Animations:** Framer Motion, Custom CSS Keyframes
* **Forms:** EmailJS integration for contact form
* **Assets:** Custom SVG Favicon & ISO headers

## 💻 Installation & Setup

Clone the repository and install dependencies to run this locally.

```bash
# 1. Clone the repository
git clone [https://github.com/kushagrakushwah/kushagra-portfolio.git](https://github.com/kushagrakushwah/kushagra-portfolio.git)

# 2. Navigate to the project directory
cd kushagra-portfolio

# 3. Install dependencies
npm install

# 4. Start the development server
npm run dev
